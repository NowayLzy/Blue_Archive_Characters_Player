#### How to use

eg:

```
ffmpeg -y -i input.mp4 -pix_fmt yuvj420p -q:v 7 -vf "transpose=1,fps=24,scale=240:320:flags=lanczos,eq=brightness=-0.05" -c:v mjpeg -an output.mjpeg
```
